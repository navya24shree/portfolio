## ğŸš€ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A code editor (VS Code, Sublime Text, etc.) - optional for development

### Installation

1. Clone the repository:
```bash
git clone https://github.com/navya24shree/portfolio.git
```

2. Navigate to the project directory:
```bash
cd portfolio
```

3. Open `index.html` in your web browser:
   - Simply double-click the `index.html` file, or
   - Use a local server (recommended for development):
     ```bash
     # Using Python
     python -m http.server 8000
     
     # Using Node.js (http-server)
     npx http-server
     ```

4. Visit `http://localhost:8000` in your browser

## ğŸ“ Customization

### Update Personal Information

1. **Profile Picture**: Replace `profile_pic.jpeg` with your own image
2. **About Section**: Edit the content in `index.html` (lines 82-115)
3. **Skills**: Modify the skill cards in the Skills section
4. **Projects**: Update project details and screenshots in the Projects section
5. **Contact Form**: Update the email address in `script.js` (line with `mailto:`)

### Update Project Screenshots

Replace the screenshot images:
- `portfolio.png` - Portfolio project screenshot
- `desktopss.png` - Currency Exchange Calculator screenshot
- `tic-tac-toess.png` - Tic Tac Toe game screenshot
- `image.png` - Campus Event Management System screenshot

### Update Colors

Modify the color scheme in `style.css` by updating the CSS variables in the `:root` selector:

```css
:root {
    --primary-color: #2563eb;
    --accent-color: #10b981;
    /* ... other colors */
}
```
